Line Plots in ggplot2

Posted by Hamza Zubair in Hitchhikers Guide to ggplot2   

This post is a part of a series of posts on Chapters from the book The Hitchhiker's guide to ggplot2 in R. The book is available for free on Leanpub.com. The data sets used in this post can be downloaded from here. Before reading this article you should have an introduction to ggplot2 grammar.¶

We are learning how to make line plots with this data. Remember from the earlier post on introduction to ggplot2 grammar, we need three things to make a plot

  1. Data (d)
  2. Aesthetic Mappings (x=year, y=export, color=product)
  3. Geometric Elements (line)

The data is in the variable d, we will map x axis to year, y axis to export and color to product. And we will use a line plot.

In [6]:
layer1 = ggplot(data=d, aes(x=year, y=export, color=product))

layer1 isn't actually much, it's basically just a blank plot.

In [7]:
layer1

To see something on the plot we need to add geometric or visual elements, like points, lines, area, text, etc...

In [10]:
layer1 + geom_point()

Obviously it's more meaningful to use lines instead of points like this...

In [11]:
layer1 + geom_line()

Notice the layered mechanism of the ggplot2 grammar. The first layer contains the data and data mappings. The points and lines above were added on the second layer. Infact we can add both layers on top of the first layer like below. And also let's call layer1 l1 for brevity...

In [21]:
l1=layer1
In [22]:
l1 + geom_line() + geom_point()

In the above graph we have three layers, the first basics layer, the line layer and then the point layer. This is what makes ggplot2 unique and powerful. Also, the ordering might not matter much in this case, since the colour of the line and the color of the points is the same, however as you shall see in the future, this fact can further help in customizing your plot.

Now, let's customize our line plot a bit. Let's make it thicker ...

In [24]:
l2=l1 + geom_line(size=1)
l2

Now we have two layers customized. Axis labels, plot titles, legends, etc. can be added as separate layers. Let's see how ...
