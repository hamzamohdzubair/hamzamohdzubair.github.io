How to Make Impressive Line Plots with ggplot2

in Hitchhikers Guide to ggplot2   

This post is a part of a series of posts on Chapters from the book The Hitchhiker's guide to ggplot2 in R. The book is available for free on Leanpub.com. The data sets used in this post can be downloaded from here. Before reading this article you should have an introduction to ggplot2 grammar.¶

We are learning how to make line plots with this data. Remember from the earlier post on introduction to ggplot2 grammar, a plot is made up of 5 components: Layer, Scale, Coordinate, Facets and Theme. Each Layer in turn is made up of five other components: data, aes, geom, stat, and position. We also saw that most of these are either advanced or have default values, leaving with the three extremely essential elements without which a plot can't exist. These three are:

  1. Data (d)
  2. Aesthetic Mappings (x-axis$\rightarrow$year, y-axis$\rightarrow$export, color$\rightarrow$product)
  3. Geometric Elements (line)

The data is in the variable d, we will map x axis to year, y axis to export and color to product. And we will use a line to represent the data. Everyone has their own styles of using ggplot2 because of its flexibility, I have mine. It's a bit different from Wickham's, the author of ggplot2, and also a bit different from Burchell and Vargas', the authors of Hitchhiker's Guide to ggplot2 in R.

In [94]:
base = ggplot(d,aes(year,export,color=product))
In [95]:
layeringline = geom_line()
In [96]:
layered = base + layeringline
In [97]:
layered

We have our core plot ready. Now we can add all sort of changes on top of it. For example, we already discussed the second component of the plot was Scale. Now, we need to change it a bit. So instead of plotting the whole thing from scratch we will just add our scale changes to the variable layered.

In [85]:
scaling1 = scale_color_manual(name="",labels=c("Copper","Pulp wood, Fruit, Salmon and others"),values=c("red","blue"))
In [86]:
scaling2 =  scale_x_continuous(breaks=2006:2014)
In [87]:
scaled = layered + scaling1 + scaling2
In [88]:
scaled

Now at this point we have a scaled plot ready. The benefit of storing everystep in graphs is that you can change any specific part without fiddlind with the others. Just for a demonstration, say I wanted to keep the change in x-axis scale, the legend labels and the change in line colors, all of which by the way is part of scaling, but change lines to points. As we already know lines are a part of layering, that is all we need to change...

In [98]:
layeringpoint = geom_point()
In [99]:
base + layeringpoint + scaling1 + scaling2

Say you need both points and lines, simple...

In [100]:
scaled + layeringpoint

I think we like this better so we will keep it...

In [101]:
scaled2 = scaled + layeringpoint

However, this is not really publishable quality, so let's work on that now. Now that we have a workable plot all we need to add to it is the theme, remember theme, it's the fifth component of a plot and this is the beautification part.

In [102]:
theminglegend =  theme(legend.position="bottom")
themingtitle = ggtitle("Composition of Exports to China ($)")
theminglabels = labs(x="Year",y="USD Million")
In [110]:
themed = scaled2 + theminglegend + themingtitle + theminglabels
themed

Once we have a basic themed plot ready, now we can do advance theming. So how about black and white...

In [113]:
themed + theme_bw()

As you can see it overwrote some of our earlier settings, specifically, the theminglegend, so will overwrite the overwriting, simply by...

In [114]:
themed + theme_bw() + theminglegend

The package ggthemes has a lot of themes. Most of them look good out-of-the-box. Some that don't, with a bit of fiddling, you are good to go. We discuss how to do the fiddling in How to make custom themes in ggplot2. Let's load ggthemes...
