The way I like to plot in ggplot2 is by using variables extensively. I like to store every single change, every single addition to a variable. The naming of the variables has a system. So we will discuss the system briefly before making the plots. Based on the grammar that we have already discussed, our plot will basically be made up of this equation:

$$plot = base + layering + scaling + coordinating + faceting + theming$$

We will also use the $ed$ variables like $layered$, $scaled$, $coordinated$, $faceted$ and $themed$ to store all the changes in the previous variables. This basically means, $$layered = base + layering$$

$$faceted = base + layering + scaling + coordinating + faceting$$$$and \thinspace so \thinspace on...$$
