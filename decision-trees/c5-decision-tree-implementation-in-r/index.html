C5 Decision Tree Implementation in R

in Decision Trees   

I would like to acknowledge the people at Edwisor for providing the training and materials for this post. For more information and detailed and engaging live sessions, please check out Edwisor.com¶

The C5.0 algorithm can be implemented in R using the C50 package. The package also comes with a data set which we will use to create decision trees.

Let's load the data...

In [47]:
library(C50)
data(churn)
d1=churnTest
d2=churnTrain
data=rbind(d1,d2)

The authors of C50 package have been so kind as to provide us with a relevant data set and also divide it into a train and test set. The division however is into $2/3^{rd}$ training set and $1/3^{rd}$ test set. We would however use only a fraction of the entire data for testing. Therefore we combined the test and train using rbind().

In [62]:
