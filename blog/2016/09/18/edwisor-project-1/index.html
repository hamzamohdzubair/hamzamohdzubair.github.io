3. Convert the required Data types¶

This is an important step in any analysis. This is one of the first things we do in Data Scrubbing (also called as Data Cleansing). Data Scrubbing is basically cleaning the data, and as I like to call it getting it ready for the main steps, in data analysis. Most of the raw data that we get comes to us in the wrong formats. As you will see below all the variables of our IMDB data are stored as Factors, which is absolutely wrong.

6. Perform binning using imdbvotes¶

Binning means to divide a numerical variable into categories so that the numeric variable may be used as a categorical variable. As we have already seen its quite trivial to convert a categorical variable into numeric, you follow: Factor $\rightarrow$ Character $\rightarrow$ Numeric. The reason you don't directly convert from factor to numeric is that you don't have direct access to the value of the variable, what you get is the index that points to the value. Converting first to character solves this problem.

Before proceeding let us reiterate basic information again: there are two types of binning:

  1. Unsupervised
  2. Supervised

Unsupervised is the simple one. It is further divided into two types:

  1. Unsupervised
    1. Equal width bins
    2. Equal frequency bins

Lets first try to bin it in equal width bins, then into equal frequency.
