Introduction¶

dplyr is one of the packages authored by Hadley Wickham, who famously also authored ggplot2. Here we will discuss how simple it is use dplyr for data manipulation. Below we will see some of the main functions that dplyr has to offer and how to use them. The list of functions includes select, filter, arrange, rename, mutate, group by and the pipeline operator. So let's load the package.

In [85]:
library(dplyr)

The data that we will use is the same data used in the book. It's available from Dr. Peng's website. You can download it from here.

In [86]:
ch=readRDS("chicago.rds")

%>% The pipeline operator¶

The operator doesn't do anything new, it just simplifies the way we use dplyr. For example, let's see the steps we took to create the last table. If you follow the variables correctly, you will notice: ch_s $\leftarrow$ ch_g $\leftarrow$ ch_m $\leftarrow$ ch_r $\leftarrow$ ch. So, the steps we performed in order are: Rename $\rightarrow$ Mutate $\rightarrow$ Group_by $\rightarrow$ Summarize. As you can see storing in separate variables can get difficult to track and cumbersome. Pipeline operator comes to our rescue.

In [116]:
rename(ch,dew=dptp,temp=tmpd,pm25=pm25tmean2,pm10=pm10tmean2,o3t=o3tmean2,no2=no2tmean2) %>%
mutate(year=as.POSIXlt(date)$year+1900) %>%
group_by(year) %>%
summarize(meanT=mean(temp,na.rm=T),maxT=max(temp,na.rm=T),minT=min(temp,na.rm=T))
