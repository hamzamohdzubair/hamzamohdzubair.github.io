Implementation of CART Decision Tree in R

in Decision Trees   

In R, CART decision trees can be created using several packages.

rpart¶

In [1]:
library(C50)
library(rpart)
library(rpart.plot)
library(rattle)

Rattle: A free graphical interface for data mining with R.
Version 4.1.0 Copyright (c) 2006-2015 Togaware Pty Ltd.
Type 'rattle()' to shake, rattle, and roll your data.

We will use the dat set that comes with the package C50.

In [2]:
data(churn)
d1=churnTest
d2=churnTrain
data=rbind(d1,d2)

It is essential to divide the data into training and test set. We will do that now...

In [3]:
set.seed(123456)
train=sample(5000,4800)
data.train=data[train,]
data.test=data[-train,]
In [4]:
tree = rpart(churn ~ . , data=data.train)
In [5]:
fancyRpartPlot(tree, sub='Decision Tree')
